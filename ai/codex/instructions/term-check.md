# Term Check Workflow

用户要求“术语巡检”“term-check”“查一下术语统一”等任务时使用本流程。

## 目标

对仓库文档做术语一致性巡检，以“术语桥”为唯一权威。

## 范围判定

- 如果用户给出聚焦术语或文件，就围绕该范围巡检。
- 如果用户未给出范围，则巡检 `docs/` 下全部 Markdown 文档。

## 执行方式

按 [terminology-guardian.md](terminology-guardian.md) 的流程执行。输出时按影响排序，每条带 `文件:行号` 与统一建议；若发现术语桥需补充新术语，单独列出。

除非用户明确要求直接修改，否则先不要改文件，等用户确认。
