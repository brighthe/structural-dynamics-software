## 一、【必须】响应谱分析算例测试报告

这是标书的一个核心打分项（★3），目前还没有可复核的算例。希望能补一份**响应谱分析算例测试报告**，建议包含三个算例：

**1. 单点激励响应谱算例**
- 模型（质量、刚度、阻尼、边界条件、响应位置）；
- 反应谱输入（谱类型、单位、阻尼比、频率/周期点）；
- 模态数据（固有频率、振型、模态参与系数）；
- 由反应谱计算各阶模态峰值并组合；
- 结果峰值表 + 与手算/解析/已有程序对比的验证结论。

**2. 多点激励响应谱算例**
- 多支座/多方向/多激励点的分组与输入设置；
- 多点激励下的模态参与与结果恢复；
- 关键节点峰值结果，并与单点结果对比；
- 验证结论。

**3. SRSS / NRL / GRP / CQC 四种叠加法对比算例**
- 同一模型、同一反应谱输入，分别算四种方法；
- 各阶模态峰值贡献表 + 四种方法组合结果对比表；
- 四种方法的公式、适用假设、参数（GRP 分组准则、CQC 模态相关系数）；
- 结果曲线/截图 + 验证结论。

> 若时间紧，至少先形成一份完整报告，包含：算例目的 → 模型说明 → 反应谱输入 → 模态结果 → 四种组合方法 → 结果与对比 → 验证结论。
