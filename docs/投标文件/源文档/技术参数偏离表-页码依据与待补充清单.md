## 2.2 【必须提供】响应谱（反应谱）分析算例测试报告

**覆盖指标：★3 具备结构响应谱分析功能 + ▲16～▲20（单点/多点激励、SRSS、NRL、GRP、CQC 叠加法）。**

**为什么列为必须：** ★3 是核心功能（★）打分项；标书第 34 页目前只是"应包含哪些材料"的材料组织说明，**没有任何可复核的响应谱算例**（模型、输入、计算过程、结果图表、验证结论均缺）。这是当前最关键、且唯一影响 ★ 核心功能的缺口。

建议**一份报告内**统一覆盖以下三个算例：

### 1. 单点激励响应谱算例（证明基本响应谱分析能力）

- **模型说明**：单自由度或多自由度基准模型，给出质量、刚度、阻尼、边界条件、自由度、响应提取位置；
- **反应谱输入**：设计反应谱或标准反应谱曲线，明确谱类型、单位、阻尼比、频率/周期点、插值方式；
- **模态数据**：固有频率、振型、模态质量或归一化方式、模态参与系数；
- **计算过程**：由反应谱值计算各阶模态峰值贡献并按指定方法组合；
- **结果输出**：位移、速度、加速度、内力或应力等指定响应量的峰值结果表；
- **验证结论**：与手算/解析/既有程序/规范算例对比，给出误差和是否满足要求的结论。

### 2. 多点激励响应谱算例（证明可组织多支座、多方向、多激励点工况）

- **支座/激励点分组**：哪些节点或支座施加响应谱输入、输入方向、是否分组；
- **多点输入设置**：各激励点反应谱、方向、比例系数、相关假设或组合规则；
- **模态参与与结果恢复**：多点激励下模态参与系数、影响向量或等效输入向量的处理；
- **组合结果**：关键节点/单元峰值响应，并说明与单点激励结果的差异；
- **验证结论**：说明多点激励施加、结果恢复和组合输出均被正确执行。

### 3. SRSS / NRL / GRP / CQC 四种叠加法对比算例（一并支撑 ▲17～▲20）

- **同一模型、同一反应谱输入**：分别计算四种方法结果，便于横向对比；
- **各阶模态峰值贡献表**：每阶模态对指定响应量的贡献值；
- **组合方法说明**：四种方法的组合公式、适用假设与参数（GRP 给出分组准则，CQC 给出模态相关系数设置）；
- **组合结果对比表**：四种方法组合峰值结果，并解释差异是否符合方法特性；
- **结果截图或曲线**：运行截图、结果表、反应谱输入曲线、组合结果图；
- **验证结论**：四种叠加法均可运行、结果可输出、差异符合工程预期。

标题：**响应谱分析算例测试报告**

3. 输入说明：反应谱曲线、阻尼比、输入方向、激励点或支座分组；
4. 模态结果：固有频率、模态参与系数、各阶模态峰值贡献；
5. 组合方法：四种方法计算口径和参数；
6. 结果与对比：组合峰值表、关键响应量结果、截图或曲线；
