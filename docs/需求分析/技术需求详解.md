## 0. 读这份文档前先建立的几个判断

### ★ vs ▲ 的技术含义
| 标记 | 含义 | 评分影响 |
|------|------|---------|
| **★** | 实质性/关键功能,不接受负偏离 | **不计分**,但不满足直接废标 |
| **▲** | 加分项,须"完全满足或正偏离" | **部分满足不得分**(全有或全无),技术分全部来自这里 |

- 技术分 = ▲ 项数 × 单项分:**A 包 26 项×2=52 / B 包 29 项×2=58 / C 包 20 项×2.5=50**。
- 大量 ▲ 项后面带括号注明**"提供算法原理描述 / 算例测试报告 / 导入导出文件 + 成功截图"**——没有证明材料 = 按负偏离 = 不得分。**证明材料的制作量,可能比算法本身的工作量还大。**

### ⚠️ 计分颗粒度差异(容易被忽略,但影响投包选择)
同样的"支持单元/连接/材料/边界"能力,在三个包里被切成的 ▲ 计分颗粒度不同:
- **A 包：细粒度**——每种单元、每种连接、每种材料、每种边界各自单列为独立 ▲ 项(共 16 项),所以 A 包 ▲ 项数最多(26)但单项分最低(2 分)。
- **B / C 包：粗粒度**——把同族合并为单条 ▲ 项(如"各向同性/各向异性/正交复材"合为 1 项,"RBAR/RBE2/RBE3"合为 1 项)。
- **后果**:同一份"单元+材料"底座实现,在 A 包能拆出十几分,在 C 包可能只折算成几条。**评估投哪个包时,要看的不是"我能不能做",而是"我做出来在该包里能折算成多少 ▲ 分"。**

### 一个值得对导师点出的背景
C 包要求的 **PEM 法(虚拟激励法,Pseudo Excitation Method)** 正是大连理工大学**林家浩**先生提出的随机振动经典方法,是 DUT 工程力学的招牌方向之一。这强烈暗示本项目与 DUT 自己的动力学/随机振动学派一脉相承——讨论时可作为"采购方真实意图"的判断依据。

## 1. 三个模块的定位与算法库构成（第三章·一、项目概况 + 二、需求清单）

三个包均**面向国产结构动力学求解器研发**,统一要求:**接口清晰、可嵌入自研国产求解器框架,具备数值稳定性、计算效率与可扩展性**。这是项目概况里反复强调的总基调——交付的不是独立软件,而是**能插进自研求解器的算法库**。

| 包 | 货物名称 | 数量 | 预算 |
|----|---------|------|------|
| A | 结构模态分析模块 | 1 套 | 220 万 |
| B | 结构动力响应分析模块 | 1 套 | 300 万 |
| C | 结构随机振动模块 | 1 套 | 210 万 |

### A 包概况：模态分析
- **算法库构成**:动力学**质量模块** + 动力学**阻尼模块** + **实特征值分析算法库** + **复特征值分析算法库**。
- **矩阵组装**:支持**一致质量矩阵与集中质量矩阵的高效组装**;能按有限元模型规模、单元类型、自由度分布完成稳定可靠的数据组织与矩阵构造。
- **阻尼能力**:覆盖**瑞利阻尼、结构阻尼、粘性阻尼**的自动计算及阻尼矩阵组装。
- **特征值**:常用实/复特征值求解算法,**支持大规模稀疏矩阵问题、复杂边界条件**。

### B 包概况：动力响应
- **算法库构成**:结构**瞬态响应**分析算法库 + 结构**频率响应**分析算法库 + **动力子结构**分析算法库。
- **求解能力**:基于**直接法和模态法**的大规模动力学方程求解,覆盖**时域与频域**动力响应。
- **频域载荷**:考虑**简谐激振力、基础运动激励**等典型频域载荷;满足稳态响应与瞬态动力学需要;支持**多工况计算组织**。
- **子结构**:适用于复杂有限元网格下的**模型缩减、子结构耦合与响应求解**。
- **对外接口**:支持**平稳随机振动频域求解调用 + 非平稳随机振动时域求解调用**——即 **B 包是 C 包的底层引擎**,要为随机振动模块预留稳定接口。

### C 包概况：随机振动
- **算法库构成**:随机振动**功率谱分析算法库** + 随机振动**时域分析算法库** + **响应谱分析算法库**。
- **激励输入**:支持**输入载荷谱定义**,处理**单点激励、多点激励**等复杂激励形式;面向大规模结构。
- **频域(功率谱)**:平稳随机振动频域求解常用算法,输出**响应功率谱、均方根**等关键结果。
- **时域**:**非平稳随机振动时域分析特色算法**,支持多工况、复杂网格。
- **统计量**:**概率密度函数、均方根响应**计算。
- **集成**:与国产自研求解器**数据接口集成**,保证"随机载荷输入→求解调用→结果提取→工程应用"流程完整可靠。

> 📌 **概况里隐含的、技术参数表没明说的硬要求**:大规模稀疏矩阵、复杂边界、多工况组织、可嵌入框架、数值稳定与可扩展。这些是"工程化/求解器级"的要求,意味着评审不只看"算法能跑通算例",还看**能否在大模型、复杂工况下稳定高效运行**。这点对从科研代码起步的团队是隐性门槛,值得在讨论中专门评估。

### 三个包的技术链关系
```
A 模态分析  ──提供模态基/特征对──▶  B 动力响应(模态法、子结构)  ──提供频域/时域求解接口──▶  C 随机振动
   (质量/阻尼矩阵、实/复特征值)        (瞬态、频响、模态综合)              (功率谱、响应谱、PEM)
```
三包共享同一套底层:**模型导入(BDF) → 单元/材料/连接/边界处理 → 质量/刚度/阻尼矩阵组装 → 求解器 → 结果导出(HDF5)**——即下一节展开的"底座"能力。

## 6. 三包核心数学公式（LaTeX）

> 按"先共性、后逐包"组织,对应第三章技术参数里的算法项。符号约定:$\mathbf{M},\mathbf{C},\mathbf{K}$ 为质量/阻尼/刚度矩阵,$\mathbf{u}$ 为位移,$\boldsymbol{\phi}_i,\omega_i,\xi_i$ 为第 $i$ 阶振型/固有频率/模态阻尼比,$\mathbf{H}(\omega)$ 为频响函数,$(\cdot)^{\mathsf H}$ 为共轭转置。

### 6.0 共性基础（三包通用）

结构动力学控制方程：

$$\mathbf{M}\ddot{\mathbf{u}}(t)+\mathbf{C}\dot{\mathbf{u}}(t)+\mathbf{K}\mathbf{u}(t)=\mathbf{F}(t)$$

单元刚度阵 / 一致(协调)质量阵 / 集中质量阵：

$$\mathbf{K}_e=\int_{\Omega_e}\mathbf{B}^{\mathsf T}\mathbf{D}\,\mathbf{B}\,\mathrm{d}\Omega,\qquad
\mathbf{M}_e^{\text{cons}}=\int_{\Omega_e}\rho\,\mathbf{N}^{\mathsf T}\mathbf{N}\,\mathrm{d}\Omega,\qquad
\mathbf{M}_e^{\text{lump}}=\operatorname{diag}\!\Big(\textstyle\sum_j (\mathbf{M}_e^{\text{cons}})_{ij}\Big)$$

瑞利阻尼（A 包阻尼模块的常用形式）：

$$\mathbf{C}=\alpha\,\mathbf{M}+\beta\,\mathbf{K}$$

### 6.1 A 包 · 模态分析

**实模态 → 广义特征值问题**（无阻尼自由振动 $\mathbf u=\boldsymbol\phi e^{i\omega t}$）：

$$\mathbf{K}\boldsymbol{\phi}_i=\omega_i^2\,\mathbf{M}\boldsymbol{\phi}_i,\qquad \lambda_i=\omega_i^2$$

振型正交性（质量归一化）：

$$\boldsymbol{\phi}_i^{\mathsf T}\mathbf{M}\boldsymbol{\phi}_j=\delta_{ij},\qquad
\boldsymbol{\phi}_i^{\mathsf T}\mathbf{K}\boldsymbol{\phi}_j=\omega_i^2\,\delta_{ij}$$

**逆迭代法 / 带位移逆迭代**（位移 $\sigma$ 加速收敛到邻近特征值）：

$$(\mathbf{K}-\sigma\mathbf{M})\,\mathbf{x}^{(k+1)}=\mathbf{M}\,\mathbf{x}^{(k)}$$

**Lanczos / Block Lanczos**：在 Krylov 子空间 $\mathcal K_m=\operatorname{span}\{\mathbf r,\mathbf A\mathbf r,\dots,\mathbf A^{m-1}\mathbf r\}$（$\mathbf A=\mathbf K^{-1}\mathbf M$）上投影,得三对角(块三对角)矩阵 $\mathbf T_m=\mathbf Q_m^{\mathsf T}\mathbf A\mathbf Q_m$,求小规模 $\mathbf T_m$ 特征值近似原问题。

**复模态 → 二次特征值问题**（有阻尼，$\lambda_i$ 复数）：

$$\big(\lambda_i^2\,\mathbf{M}+\lambda_i\,\mathbf{C}+\mathbf{K}\big)\boldsymbol{\phi}_i=\mathbf{0}$$

线性化为 $2n$ 维状态空间广义特征值问题（对称形式）：

$$(\lambda\,\mathbf{A}+\mathbf{B})\,\mathbf{z}=\mathbf 0,\quad
\mathbf{A}=\begin{bmatrix}\mathbf{C}&\mathbf{M}\\\mathbf{M}&\mathbf 0\end{bmatrix},\
\mathbf{B}=\begin{bmatrix}\mathbf{K}&\mathbf 0\\\mathbf 0&-\mathbf{M}\end{bmatrix},\
\mathbf{z}=\begin{bmatrix}\boldsymbol\phi\\\lambda\boldsymbol\phi\end{bmatrix}$$

### 6.2 B 包 · 动力响应

**频率响应（FRF）**：简谐激励 $\mathbf F=\hat{\mathbf F}e^{i\omega t}$，稳态 $\mathbf u=\hat{\mathbf u}e^{i\omega t}$：

$$\big(\mathbf{K}-\omega^2\mathbf{M}+i\omega\mathbf{C}\big)\hat{\mathbf u}=\hat{\mathbf F}
\;\Rightarrow\;
\hat{\mathbf u}=\mathbf{H}(\omega)\hat{\mathbf F},\quad
\mathbf{H}(\omega)=\big(\mathbf{K}-\omega^2\mathbf{M}+i\omega\mathbf{C}\big)^{-1}$$

**模态法频响**（用前 $m$ 阶振型展开 $\mathbf H$，避免逐频率求逆）：

$$\mathbf{H}(\omega)=\sum_{i=1}^{m}\frac{\boldsymbol{\phi}_i\boldsymbol{\phi}_i^{\mathsf T}}{\omega_i^2-\omega^2+2i\xi_i\omega_i\omega}$$

**模态叠加（瞬态）**：$\mathbf u=\boldsymbol\Phi\mathbf q$，解耦为单自由度方程：

$$\ddot q_i+2\xi_i\omega_i\,\dot q_i+\omega_i^2\,q_i=\boldsymbol{\phi}_i^{\mathsf T}\mathbf F(t)$$

**直接积分 — Newmark 族**（Generalized-α 的基础）：

$$\mathbf u_{n+1}=\mathbf u_n+\Delta t\,\dot{\mathbf u}_n+\tfrac{\Delta t^2}{2}\big[(1-2\beta)\ddot{\mathbf u}_n+2\beta\,\ddot{\mathbf u}_{n+1}\big]$$
$$\dot{\mathbf u}_{n+1}=\dot{\mathbf u}_n+\Delta t\big[(1-\gamma)\ddot{\mathbf u}_n+\gamma\,\ddot{\mathbf u}_{n+1}\big]$$

**Generalized-α**：在中间时刻 $t_{n+1-\alpha}$ 处配置残差（$\alpha_m,\alpha_f$ 控制高频数值阻尼）：

$$\mathbf{M}\ddot{\mathbf u}_{n+1-\alpha_m}+\mathbf{C}\dot{\mathbf u}_{n+1-\alpha_f}+\mathbf{K}\mathbf u_{n+1-\alpha_f}=\mathbf F(t_{n+1-\alpha_f})$$

**Wilson-θ**：在 $[t_n,\,t_n+\theta\Delta t]$ 内假设加速度线性变化（$\theta\ge1.37$ 保证无条件稳定）。
**Bathe**：单步分两子步（先梯形法、后三点后向差分），组合获得耗散与稳定性。

**动态子结构 — 固定界面模态综合法（Craig–Bampton）**：自由度分为界面 $b$ 与内部 $i$，

$$\begin{bmatrix}\mathbf u_b\\\mathbf u_i\end{bmatrix}
=\underbrace{\begin{bmatrix}\mathbf I&\mathbf 0\\\boldsymbol\Psi_{ib}&\boldsymbol\Phi_{ik}\end{bmatrix}}_{\mathbf T}
\begin{bmatrix}\mathbf u_b\\\mathbf q_k\end{bmatrix},\qquad
\boldsymbol\Psi_{ib}=-\,\mathbf K_{ii}^{-1}\mathbf K_{ib}\ \text{(约束模态)},\quad
\mathbf K_{ii}\boldsymbol\Phi_{ik}=\mathbf M_{ii}\boldsymbol\Phi_{ik}\boldsymbol\Lambda_k\ \text{(固定界面主模态)}$$

缩聚后子结构矩阵：$\tilde{\mathbf M}=\mathbf T^{\mathsf T}\mathbf M\,\mathbf T,\ \ \tilde{\mathbf K}=\mathbf T^{\mathsf T}\mathbf K\,\mathbf T$。

### 6.3 C 包 · 随机振动

**平稳随机振动 — 响应功率谱**（输入功率谱 $\mathbf S_{FF}$，输出 $\mathbf S_{uu}$）：

$$\mathbf{S}_{uu}(\omega)=\mathbf{H}(\omega)\,\mathbf{S}_{FF}(\omega)\,\mathbf{H}^{\mathsf H}(\omega)$$

**响应均方根（RMS）= 功率谱曲线下面积的平方根**（$m$ 阶谱矩 $\lambda_m=\int_0^\infty\omega^m S_{uu}\,\mathrm d\omega$）：

$$\sigma_u^2=\int_{-\infty}^{\infty}S_{uu}(\omega)\,\mathrm d\omega,\qquad u_{\mathrm{rms}}=\sigma_u$$

**PEM 虚拟激励法（DUT 林家浩，C 包招牌）**：把单源完全相干谱写成 $\mathbf S_{FF}=\mathbf s\,\mathbf s^{\mathsf H}$，构造虚拟激励 $\tilde{\mathbf F}=\mathbf s\,e^{i\omega t}$，做一次确定性简谐响应 $\tilde{\mathbf u}=\mathbf H\mathbf s\,e^{i\omega t}$，则响应谱精确等于：

$$\mathbf{S}_{uu}(\omega)=\tilde{\mathbf u}^{*}\,\tilde{\mathbf u}^{\mathsf T}=(\mathbf H\mathbf s)^{*}(\mathbf H\mathbf s)^{\mathsf T}$$

多点部分相干时，先分解 $\mathbf S_{FF}=\sum_k\mathbf s_k\mathbf s_k^{\mathsf H}$（特征/Cholesky），叠加 $\mathbf S_{uu}=\sum_k(\mathbf H\mathbf s_k)^{*}(\mathbf H\mathbf s_k)^{\mathsf T}$。**核心价值：把随机振动谱计算转化为一系列确定性简谐分析，效率极高。**

**非平稳随机振动 — 演变功率谱**（调制函数 $A(\omega,t)$，时变方差）：

$$\tilde{\mathbf F}(\omega,t)=A(\omega,t)\,\mathbf s\,e^{i\omega t},\qquad
\sigma_u^2(t)=\int_{-\infty}^{\infty}\big|\tilde u(\omega,t)\big|^2\,\mathrm d\omega$$

**响应谱法 — 振型组合**（由反应谱值得各阶峰值响应 $r_i$，再组合峰值 $R$）：

$$\text{SRSS:}\ R=\sqrt{\sum_i r_i^2}
\qquad
\text{CQC:}\ R=\sqrt{\sum_i\sum_j \rho_{ij}\,r_i\,r_j}$$

CQC 相关系数（$r=\omega_j/\omega_i$）：

$$\rho_{ij}=\frac{8\sqrt{\xi_i\xi_j}\,(\xi_i+r\,\xi_j)\,r^{3/2}}
{(1-r^2)^2+4\xi_i\xi_j\,r\,(1+r^2)+4(\xi_i^2+\xi_j^2)\,r^2}$$

$$\text{NRL:}\ R=|r_k|+\sqrt{\sum_{i\ne k} r_i^2}\ \ (k=\arg\max_i|r_i|)
\qquad
\text{GRP:}\ \text{邻近模态分组内绝对值相加、组间 SRSS}$$
