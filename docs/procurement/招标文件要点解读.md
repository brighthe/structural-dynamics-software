## 四、技术需求核心（第三章 用户需求书）★最值得和导师深入讨论★

三个包均"面向国产结构动力学求解器研发"，要求模块**接口清晰、可嵌入自研国产求解器框架**，并**交付全部可编译源代码、知识产权归采购人**。技术指标分 ★/▲ 两类：

> - **★ 关键指标**：不接受负偏离，未响应或负偏离 → **投标无效**；但 ★ **不计入技术评分**。
> - **▲ 加分项**：完全满足或正偏离才得分，**部分满足不得分**；多数 ▲ 项须附"算法原理描述 + 算例测试报告 / 导入导出文件 + 截图"，否则按负偏离（不得分）。

| 包 | 模块 | 预算 | ▲ 项 / 技术分 | 核心算法关键词 |
|----|------|:----:|:-----------:|--------------|
| A | 结构模态分析 | 220 万 | 26 项 / 52 分 | 实模态（逆迭代、Lanczos / Block Lanczos）、复模态 |
| B | 结构动力响应 | 300 万 | 29 项 / 58 分 | 瞬态（Wilson-θ / Generalized-α / Bathe）、频响（直接 / 模态法）、模态综合 |
| C | 结构随机振动 | 210 万 | 20 项 / 50 分 | 功率谱（CQC / SRSS / **PEM**）、响应谱（SRSS / NRL / GRP / CQC）、非平稳 |

三包共享同一底座：BDF 导入 → 单元 / 材料 / 连接 / 边界处理 → 质量 / 刚度 / 阻尼矩阵组装 → HDF5 导出。

> 📌 **逐项 ★/▲ 指标拆解、核心数学公式（LaTeX）、与拓扑优化背景的知识对应，详见 [技术需求详解.md](技术需求详解.md)。**

## 七、需要和导师讨论 / 待澄清的问题
