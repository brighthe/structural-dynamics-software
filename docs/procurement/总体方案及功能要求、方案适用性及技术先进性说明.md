## 一、项目理解与需求分析

### 1.1 项目背景与采购本质

本项目面向**国产结构动力学求解器自主研发**，C 包采购“结构随机振动分析模块”一套。与采购商业软件授权根本不同，招标文件明确要求：**中标人交付软件模块及可编译该模块的全部源代码，自交付起软件模块及全部代码的知识产权归采购人所有**。这决定了 C 包的本质是“随机振动算法库委托研发 + 源代码与知识产权完整移交”，对投标人的要求不是“有一套能用的软件”，而是：

（1）**算法自主可控**——随机振动全部核心算法（功率谱、非平稳时域、响应谱）具备完整的数学原理依据与自主代码实现，不依赖任何受限的第三方商业库；

（2）**代码可移交、可维护**——源代码工程化、文档化，采购人接手后能够独立编译、修改、扩展与二次开发；

（3）**可嵌入自研框架**——模块接口清晰、数据协议明确，能够嵌入采购人自研的国产结构动力学求解器框架，并支持与国产软硬件平台集成。

### 1.2 C 包技术任务的本质理解

招标文件第三章对 C 包的定位是：“面向国产结构动力学求解器研发，提供结构随机振动功率谱分析算法库、结构随机振动时域分析算法库和响应谱分析算法库。”我方对其技术本质的理解如下：

确定性动力分析回答“给定载荷下结构怎么动”，而**随机振动回答“在统计描述的随机激励下，结构响应的统计特征是什么”**——火箭发射段的宽频随机载荷、路面对车辆的随机激励、地震对结构的随机/反应谱激励，其输入本身无法用确定的时间函数描述，只能用**功率谱密度**或**反应谱**刻画。C 包要交付的，正是把“随机激励的统计描述”映射为“结构响应的统计描述”的三类算法：

（1）**平稳随机振动（频域）**——已知输入功率谱矩阵 $\mathbf S_{FF}(\omega)$，求响应功率谱 $\mathbf S_{uu}(\omega)$ 与响应标准差 $\sigma_u$；对于零均值随机响应，标准差等于均方根（RMS）。核心是频响函数 $\mathbf H(\omega)$ 与谱关系 $\mathbf S_{uu}=\mathbf H\,\mathbf S_{FF}\,\mathbf H^{\mathsf H}$；

（2）**非平稳随机振动（时域）**——输入统计特征随时间变化（如发射、冲击的调制随机过程），求时变标准差 $\sigma_u(t)$、时变均方根 $u_{\mathrm{rms}}(t)$ 与固定时刻的响应边缘概率密度函数 $p_u(x,t)$；

（3）**响应谱分析**——已知设计反应谱、结构模态和阻尼等数据，用模态叠加估计结构峰值响应 $R$（由各阶模态峰值贡献 $r_i$ 组合而成）。

### 1.3 C 包与 A、B 包的关系及在整体求解器中的定位

本项目 A、B、C 三包面向同一个国产结构动力学求解器，共享同一套有限元建模底座（混合网格 → 单元/材料/连接/边界 → $\mathbf M$/$\mathbf C$/$\mathbf K$ 组装），并在数据上层层衔接，构成“模态 → 确定性响应 → 随机响应”的完整动力学分析链：

- **A 包（结构模态分析）**——求结构的固有频率与振型，是整个动力学分析体系的地基；
- **B 包（结构动力响应分析）**——在模态基础上求确定性载荷（瞬态/频响）下的响应，且招标文件明确要求其“支持平稳随机振动频域求解调用和非平稳随机振动时域求解调用，为后续随机振动分析模块提供稳定接口”——即 B 包在设计上即为 C 包预留了求解调用入口；
- **C 包（结构随机振动分析，本投标包）**——处理随机/反应谱激励下的统计响应，位于这条分析链的最上层：其 CQC/SRSS 功率谱与响应谱四种叠加法以**模态数据（A 包能力）**为输入，其 PEM 与非平稳时域分析以**频域/时域求解能力（B 包能力）**为后端。

也就是说，在采购人最终的整体求解器蓝图中，**C 包自然地以 A 包的模态数据与 B 包的求解能力为上游**。我方对这一关系的应对策略分别面向“C 包独立交付验收”与“采购人整体集成”两种情形：

（1）**C 包独立交付验收时**——C 模块提供标准化数据与求解服务接口，可接收质量矩阵、阻尼矩阵、刚度矩阵、模态数据以及频域/时域求解结果；同时配套标准算例、基准数据和必要的测试支撑组件，确保 C 包核心算法可独立运行、验证和验收，不以 A、B 包是否同时交付为前提（详见 1.4）；

（2）**面向整体集成时**——C 模块的数据接口与 A、B 包的自然边界保持一致：模态数据接口承接 A 包输出，频域/时域求解服务接口承接 B 包为随机振动预留的调用能力。具体数据结构、调用协议和集成边界在项目需求确认与接口设计阶段与采购人共同固化，以降低后续适配工作量，避免接口孤岛与重复开发。

这一设计既保证 C 包核心算法作为独立标的可验证、可验收，又使其能够复用采购人现有能力或 A、B 包成果，契合采购人“三模块协同、统一求解器”的总体目标。

### 1.4 C 包支撑能力与接口边界

C 包三类分析算法需要相应的结构动力学基础数据与求解能力：

## 二、总体方案

### 2.1 随机振动分析的理论基础

本节建立三类随机振动算法共用的理论根基——随机激励的统计描述，以及结构统计响应如何由频响函数与模态导出，为下文总体方案与第三章功能响应提供统一的原理依据。

#### 2.1.1 随机激励的统计描述与谱表示

承接 1.2，确定性动力分析处理"给定时间历程 $\mathbf F(t)$ 下的响应"，而工程中大量激励——火箭发射段宽频噪声、路面对车辆的随机起伏、地震地面运动——其时间历程本身不可预测，只能用统计量刻画。对零均值平稳随机激励，最常用的二阶统计描述是自/互相关函数 $R_{F_aF_b}(\tau)=\mathrm E\!\left[F_a(t)F_b(t+\tau)\right]$，其频域对偶即功率谱密度，二者由维纳–辛钦（Wiener–Khinchin）关系联系：

$$S_{F_aF_b}(\omega)=\frac{1}{2\pi}\int_{-\infty}^{+\infty}R_{F_aF_b}(\tau)\,e^{-i\omega\tau}\,\mathrm d\tau$$

当 $a=b$ 时为自功率谱（非负实函数），$a\ne b$ 时为互功率谱（复函数，其模与相位刻画两激励分量的相关强度与相位差）。各分量组装为 Hermitian 半正定的输入功率谱矩阵 $\mathbf S_{FF}(\omega)$。对地震等以反应谱给定的激励，则不直接给出 $\mathbf S_{FF}$，而由设计反应谱描述单自由度体系峰值响应随频率的包络（见 2.1.5）。本方案统一采用双边角频率谱约定，方差由全频域积分得到（具体见 ▲13）；若采用单边谱或其他傅里叶变换约定，则同步调整积分区间与归一化系数<sup>[7,8]</sup>。

#### 2.1.2 平稳响应的输入–输出谱关系（CQC/SRSS 之根）

对线性时不变结构，运动方程 $\mathbf M\ddot{\mathbf u}+\mathbf C\dot{\mathbf u}+\mathbf K\mathbf u=\mathbf F(t)$ 在频域的解由频响函数

$$\mathbf H(\omega)=(\mathbf K-\omega^2\mathbf M+i\omega\mathbf C)^{-1}$$

给出，即 $\mathbf u(\omega)=\mathbf H(\omega)\mathbf F(\omega)$。代入功率谱定义并利用激励的二阶统计量，可得平稳随机振动的核心谱关系：

$$\mathbf S_{uu}(\omega)=\mathbf H(\omega)\,\mathbf S_{FF}(\omega)\,\mathbf H^{\mathsf H}(\omega)$$

这是把"激励的统计描述"映射为"响应的统计描述"的基本桥梁<sup>[7,9]</sup>。对大型结构逐频点直接求逆 $\mathbf H$ 代价高昂，工程上改用模态叠加：设保留前若干阶模态 $\boldsymbol\phi_i$ 及模态频响 $H_i(\omega)$，则指定自由度的响应自谱展开为对模态对 $(i,j)$ 的双重求和，含 $H_i(\omega)H_j^*(\omega)$ 因子。其中 $i\ne j$ 的**交叉项来自不同模态响应之间的统计相关**：当固有频率密集、阻尼较大时交叉项不可忽略，须完整保留——这正是 CQC（完全二次组合）的依据<sup>[4,5]</sup>；当固有频率分离良好、模态相关较弱时交叉项趋于零，仅保留 $i=j$ 项即得 SRSS，二者结果趋于一致。两者完整的模态叠加公式与适用条件见 ▲10、▲11。

#### 2.1.3 虚拟激励法（PEM）的数学本质

虚拟激励法（Pseudo Excitation Method, PEM）是本方案功率谱与非平稳分析的核心算法，其要点是把随机响应谱的计算**严格转化**为一组确定性简谐响应的计算。数学依据为：Hermitian 半正定的输入谱矩阵可逐频点分解为 $\mathbf S_{FF}(\omega)=\sum_k\mathbf s_k(\omega)\,\mathbf s_k^{\mathsf H}(\omega)$（特征值分解或带主元 Cholesky；其中 $\mathbf s_k$ 已含特征值平方根缩放，即特征值分解下 $\mathbf s_k=\sqrt{\lambda_k}\,\boldsymbol\psi_k$，$\lambda_k\ge 0$ 因 $\mathbf S_{FF}$ 半正定）。对每个谱分量构造确定性虚拟简谐激励 $\tilde{\mathbf F}_k=\mathbf s_k\,e^{i\omega t}$，其确定性响应为 $\tilde{\mathbf u}_k=\mathbf H\,\mathbf s_k$，于是

$$\mathbf S_{uu}(\omega)=\sum_k\tilde{\mathbf u}_k\,\tilde{\mathbf u}_k^{\mathsf H}=\mathbf H\Big(\sum_k\mathbf s_k\,\mathbf s_k^{\mathsf H}\Big)\mathbf H^{\mathsf H}=\mathbf H\,\mathbf S_{FF}\,\mathbf H^{\mathsf H}$$

即与 2.1.2 的谱关系完全等价，且**自动、精确地保留了多点激励的相关性与模态交叉贡献**，无需显式展开双重求和的交叉项。其效率来源在于：所需确定性频响计算次数等于有效谱分量数（即 $\mathbf S_{FF}$ 的秩），当输入谱低秩或有效激励通道数远少于保留模态数时，计算量显著低于完整 CQC 交叉项展开。

虚拟激励法由**大连理工大学林家浩教授**于 1980 年代中期提出，并发展为平稳/非平稳随机振动、随机地震响应、车桥耦合等工程问题的通用高效算法<sup>[1,2,3]</sup>，在国内外结构动力学界广泛采用。本方案直接继承这一源自本校、国际领先的随机振动算法体系，作为 ▲12（PEM 功率谱）与 ▲14（非平稳时变均方根）的算法内核。

#### 2.1.4 非平稳随机振动与演变谱

火箭发射、爆炸冲击、地震动等激励的统计特征随时间显著变化，属非平稳随机过程，不能用与时间无关的 $\mathbf S_{FF}(\omega)$ 描述。本方案采用 Priestley 的**演变功率谱**（evolutionary power spectrum）$\mathbf G_{FF}(\omega,t)$ 刻画其时变频率含量<sup>[10]</sup>。工程中常用的标量调制模型为 $\mathbf G_{FF}(\omega,t)=|A(\omega,t)|^2\,\mathbf S_{FF}^0(\omega)$，其中 $\mathbf S_{FF}^0$ 为基准平稳谱、$A(\omega,t)$ 为时变标量调制（包络）函数；当调制函数与频率无关（$A=A(t)$）时即退化为均匀调制（uniformly modulated）模型。该模型以单一标量调制全谱，用于描述调制型非平稳过程，不代表需要矩阵调制的所有一般非平稳过程。

关键之处在于：**非平稳响应不能把平稳谱关系逐时刻照搬**。线性结构对激励存在"记忆"（瞬态响应随阻尼衰减），某时刻的响应统计量取决于该时刻之前的整个激励历程，而非仅当前瞬时谱。故本方案采用**时域虚拟激励法**<sup>[3]</sup>：将基准谱分解后构造时变虚拟激励 $\tilde{\mathbf F}_k(\omega,t)=A(\omega,t)\,\mathbf s_k(\omega)\,e^{i\omega t}$，经时域逐步积分求得 $\tilde{\mathbf u}_k(\omega,t)$，再重构响应演变谱 $\mathbf G_{uu}(\omega,t)=\sum_k\tilde{\mathbf u}_k\,\tilde{\mathbf u}_k^{\mathsf H}$，由其得到时变均方根与响应概率密度（见 ▲14、▲15）。

#### 2.1.5 响应谱法与模态组合的统计基础

地震等激励在工程规范中常以**设计反应谱** $S(\omega,\xi)$ 给定——它是单自由度体系在该激励下峰值响应随固有频率、阻尼比变化的包络曲线。响应谱法借此估计多自由度结构的峰值响应：先求各阶模态参与系数 $\Gamma_i$ 与该阶峰值贡献 $r_i$（推导见 3.2 节（四）开头），再对各阶峰值组合得总峰值估计 $R$。

组合的统计实质是：各阶模态峰值通常**不在同一时刻发生**，直接代数相加过于保守，直接取最大值又偏不安全，故须按其统计相关性组合。四种组合法对应不同假设——SRSS 假设各阶峰值响应互不相关（统计不相关，频率分离良好）<sup>[4]</sup>；CQC 以相关系数 $\rho_{ij}$ 计入密集模态相关，是 SRSS 在相关情形下的推广<sup>[4,5]</sup>；NRL（美国海军研究实验室法）<sup>[11]</sup> 与 GRP（分组法）<sup>[6]</sup> 则为工程规范中分别面向保守设计与密频分组的实用组合规则。四法的公式、适用范围与规范配置见 ▲17–▲20。

#### 2.1.6 小结

三类算法同根于"统计输入 → 统计输出"的映射：平稳分析经频响/模态谱关系（2.1.2）或 PEM（2.1.3），非平稳分析经演变谱与时域虚拟激励（2.1.4），响应谱分析经模态参与与统计组合（2.1.5）。下文总体方案即把这套理论组织为"标准支撑接口 + 三类算法库 + 测试支撑组件"的工程实现，第三章再逐条落实为对 ★/▲ 指标的响应。

### 2.2 总体技术路线

C 模块采用“**一套标准支撑接口 + 三类算法库 + 一组测试支撑组件**”的总体技术路线：

**一套标准支撑接口**——面向采购人总体求解器定义统一的数据与服务边界，接收模型及载荷数据、$\mathbf M$/$\mathbf C$/$\mathbf K$ 矩阵、模态数据，并调用频率响应和时间积分服务；接口同时支持采购人现有框架、A/B 模块及测试支撑组件接入。

**三类算法库**——对应招标 C 包要求的三类算法：

（1）**功率谱分析算法库**：CQC、SRSS、PEM（虚拟激励法）求响应功率谱，及响应均方根计算；

（2）**随机振动时域分析算法库**：非平稳随机振动的时变均方根与响应概率密度函数计算；

（3）**响应谱分析算法库**：单点/多点激励下 SRSS、NRL、GRP、CQC 四种模态组合的峰值响应 $R$ 估计。

**一组测试支撑组件**——提供标准模型、矩阵与模态基准数据、频域/时域模拟服务、算例驱动和结果比对工具，用于脱离 A、B 模块开展算法测试、回归验证和独立验收。

技术路线的关键决策：

**（1）标准化模态数据作为枢纽。** CQC/SRSS 功率谱和响应谱四种叠加法共享统一的模态数据结构（频率、振型、模态质量/刚度/阻尼、模态参与系数），支持 A 包、采购人现有模态模块或测试基准数据接入，实现一次计算、多分析复用。

**（2）频响服务与 PEM 深度结合。** PEM 将随机响应谱计算转化为一系列确定性简谐响应计算。C 模块通过统一频响服务接口批量提交虚拟激励并获取响应，使其能够复用 B 包或采购人现有频率响应能力；独立测试时则由测试支撑组件提供基准频响数据或轻量化验证服务。

**（3）面向大规模数据的接口设计。** 矩阵和谱数据接口支持稀疏存储、批量频点、分块输出及按需提取，避免不必要的数据复制；具体模态、频响和时域求解算法由所接入的 A/B 模块或采购人求解服务负责。

**（4）多工况组织。** 激励谱、反应谱、约束集合以“工况（Case）”对象组织，支持单次建模、多工况批量求解与分组导出，满足招标对多工况计算组织的要求。

模块采用自下而上四层架构，各层接口明确、可独立测试：

**Layer 0 —— 数据协议与适配层。**

（1）定义模型标识、自由度映射、矩阵、模态、随机激励谱、反应谱、工况及结果数据协议，并提供输入完整性、维度一致性、单位和频率网格校验；

| 验证对象 | 参照基准 | 量化指标 | 拟定通过阈值 |
|---|---|---|---|
| 模态基础数据 | 解析解或基准模型 | 固有频率相对误差 | ≤ 1% |
| 平稳随机振动功率谱（▲10–▲12） | SDOF 闭式解、基准频响 | 共振频率误差 / 谱峰值相对误差 | ≤ 1% / ≤ 5% |

| 响应谱组合（▲16–▲20） | 线性时程峰值包络 | 关键点峰值相对误差 | 一般工况 ≤ 10–15% |
| 自动回归 | 历史版本基准结果集 | 关键算例通过率 | 100% |

上述阈值依据随机振动分析的常规误差来源（模态截断、频率离散、积分收敛、样本统计涨落）拟定；具体数值在合同与验收方案确认阶段与采购人共同固化。

### 3.1 ★ 核心功能逐条响应（3 项，全部完全满足）

**★1 具备结构平稳随机振动分析功能。** 响应：**完全满足，无偏离**。面向线性结构的平稳随机振动分析，以输入激励功率谱矩阵

$$\mathbf S_{FF}(\omega)=\left[S_{F_aF_b}(\omega)\right]$$

描述已映射至结构物理自由度空间的各输入载荷分量的谱强度及相关关系。其中，$S_{F_aF_b}(\omega)$ 表示结构物理自由度上的输入载荷分量 $a$ 与 $b$ 在频率 $\omega$ 处的互功率谱，用于描述两载荷分量的频域相关强度和相位关系；当 $a=b$ 时为自功率谱。通过频响函数

$$\mathbf H(\omega)=(\mathbf K-\omega^2\mathbf M+i\omega\mathbf C)^{-1}$$

计算响应功率谱矩阵

$$\mathbf S_{uu}(\omega)=\mathbf H(\omega)\,\mathbf S_{FF}(\omega)\,\mathbf H^{\mathsf H}(\omega)$$

其中，$\mathsf H$ 表示共轭转置。谱关系 $\mathbf S_{uu}=\mathbf H\,\mathbf S_{FF}\,\mathbf H^{\mathsf H}$ 的推导见 2.1.2<sup>[7,9]</sup>。提供 CQC（完全二次组合，Complete Quadratic Combination）、SRSS（平方和开方，Square Root of the Sum of Squares）、PEM（虚拟激励法，Pseudo Excitation Method）三种响应功率谱求解算法，输出响应功率谱和响应标准差；对于零均值随机响应，标准差等于均方根。

**★2 具备结构非平稳随机振动分析功能。** 响应：**完全满足，无偏离**。面向线性结构，支持均匀调制及采用演变功率谱描述的非平稳随机激励。总体计算关系为

$$\mathbf G_{FF}(\omega,t)\xrightarrow{\text{时域虚拟激励与逐步积分}}\mathbf G_{uu}(\omega,t)\xrightarrow{\text{统计后处理}}\left\{\sigma_u(t),\,p_u(x,t)\right\}$$

其中，$\mathbf G_{FF}(\omega,t)$ 和 $\mathbf G_{uu}(\omega,t)$ 分别为输入与响应演变功率谱矩阵，$\sigma_u(t)$ 为时变响应标准差；零均值条件下等于时变均方根。在线性高斯激励条件下，根据响应均值和方差计算固定时刻的边缘概率密度；对于线性系统下的非高斯激励，在已给定输入概率模型、样本数据或样本生成规则的条件下，采用样本统计方法估计概率密度。

**★3 具备结构响应谱分析功能。** 响应：**完全满足，无偏离**。以设计反应谱、结构固有频率、振型、模态阻尼比、输入方向及模态参与系数为输入，计算各阶模态对指定响应量的峰值贡献，并进行模态组合：

$$\left\{S(\omega_i,\xi_i),\,\boldsymbol\phi_i,\,\Gamma_i\right\}\longrightarrow r_i\xrightarrow{\mathrm{SRSS/NRL/GRP/CQC}}R$$

其中，$S(\omega_i,\xi_i)$ 表示在第 $i$ 阶固有频率和阻尼比处取得的设计反应谱值，$r_i$ 为该阶模态对指定响应量的峰值贡献，$R$ 为组合后的总峰值估计。支持单点及多点激励，提供 SRSS、NRL、GRP、CQC 四种模态组合方法。

#### （二）响应功率谱算法（▲10–▲13，完全响应）

**▲10 支持 CQC 算法求解响应功率谱。** 完全响应。在线性、经典阻尼、实模态及平稳随机激励条件下，采用模态叠加计算响应功率谱，并保留截断模态集合内的全部模态自谱项和交叉谱项。本节中的 $\mathbf S_{FF}(\omega)$ 均指已映射至结构物理自由度空间的输入激励功率谱矩阵。设第 $i$ 阶模态频响 $H_i(\omega)=\big(\omega_i^2-\omega^2+2i\xi_i\omega_i\omega\big)^{-1}$（单位模态质量归一化），则模态（广义）力互谱为

$$S_{f_if_j}(\omega)=\boldsymbol\phi_i^{\mathsf T}\mathbf S_{FF}(\omega)\boldsymbol\phi_j$$

指定自由度处的响应自功率谱为

$$S_{uu}(\omega)=\sum_{i}\sum_{j}\phi_{pi}\,\phi_{pj}\,H_i(\omega)\,H_j^*(\omega)\,S_{f_if_j}(\omega)$$

其中，$\boldsymbol\phi_i$ 为第 $i$ 阶振型向量，$\phi_{pi}$ 为其在指定响应自由度 $p$ 处的分量。该方法考虑模态间交叉相关影响，适用于密集模态结构（交叉项的来源见 2.1.2<sup>[4,5]</sup>）；计算精度同时取决于模态截断范围、阻尼模型、频率离散及输入谱数据。此处 CQC 指功率谱分析中的完全二次模态谱组合，不同于 ▲20 响应谱分析中采用相关系数 $\rho_{ij}$ 组合各阶峰值响应的 CQC 方法。

**▲11 支持 SRSS 算法求解响应功率谱。** 完全响应。在 ▲10 模态叠加基础上**忽略模态交叉项（仅保留 $i=j$ 项）**：

$$S_{uu}(\omega)=\sum_{i}\phi_{pi}^2\,|H_i(\omega)|^2\,S_{f_if_i}(\omega)$$

忽略交叉项的成立条件是各阶共振峰在频率轴上分离良好：模态频响 $H_i(\omega)$ 的峰宽由阻尼决定，当相邻固有频率间距远大于峰宽时，$i\ne j$ 的 $H_i(\omega)H_j^*(\omega)$ 幅值不会同时接近各自峰值，交叉项对响应谱及其频率积分的贡献可忽略。故 SRSS 适用于固有频率分离较好、模态响应相关性较弱的结构，计算高效；在上述条件下其结果与 ▲10 的 CQC 趋于一致，而频率密集时应改用 CQC，以免低估模态交叉相关。

**▲12 支持 PEM 法求解响应功率谱算法。**（提供算法原理描述与算例测试报告）完全响应。**虚拟激励法（Pseudo Excitation Method）** 由大连理工大学林家浩教授提出并发展为通用高效算法<sup>[1,2,3]</sup>，其等价性与效率分析见 2.1.3。该法将 Hermitian 半正定的激励功率谱矩阵逐频点采用特征值分解或带主元 Cholesky 等稳健方法分解为 $\mathbf S_{FF}(\omega)=\sum_k \mathbf s_k(\omega)\,\mathbf s_k^{\mathsf H}(\omega)$。对每个已定义在结构物理自由度空间的谱分量构造确定性虚拟简谐激励 $\tilde{\mathbf F}_k(\omega,t)=\mathbf s_k(\omega)\,e^{i\omega t}$，求其确定性简谐响应 $\tilde{\mathbf u}_k(\omega)=\mathbf H(\omega)\,\mathbf s_k(\omega)$，则响应功率谱

$$\mathbf S_{uu}(\omega)=\sum_k \tilde{\mathbf u}_k(\omega)\,\tilde{\mathbf u}_k^{\mathsf H}(\omega)$$

在所采用频响模型、谱分解精度和模态截断范围内，PEM 保留输入通道相关性及相应的模态交叉贡献，将随机响应谱计算转化为一组确定性频响计算。所需确定性频响计算次数等于保留的有效谱分量数；当输入谱矩阵低秩或有效激励通道数明显少于保留模态数时，可较完整模态交叉项展开显著降低计算量。提供算法原理描述与算例测试报告，并与 CQC 或解析解进行交叉校核。

**▲13 支持响应均方根计算。**（提供算例测试报告）完全响应。采用“响应功率谱沿双边角频率积分直接得到方差”的谱归一化约定，指定响应分量的方差为

$$\sigma_u^2=\int_{-\infty}^{+\infty} S_{uu}(\omega)\,\mathrm d\omega$$

响应均方根为 $u_{\mathrm{rms}}=\sqrt{\mu_u^2+\sigma_u^2}$；对于通常采用的零均值随机响应，$u_{\mathrm{rms}}=\sigma_u$。若采用其他傅里叶变换或单边谱约定，则同步调整积分区间和归一化系数。输出节点/单元指定响应量的均方根云图数据；在共振峰附近进行自适应频率网格加密以提高积分分辨率，并依据积分收敛准则控制数值误差。提供算例测试报告（采用具有物理质量 $m$、固有频率 $\omega_n$、阻尼比 $\xi$ 的单自由度系统作为闭式校核基准，与 ▲10–▲12 模态叠加所采用的单位模态质量归一化频响相互独立；在力功率谱 $S_0$（双边白噪声）激励下，其位移方差解析解为 $\sigma_u^2=\dfrac{\pi S_0}{2\xi\omega_n^3 m^2}$）。

#### （三）非平稳随机振动算法（▲14–▲15，完全响应）

**▲14 非平稳随机振动分析支持时变响应均方根计算。**（提供算例测试报告）完全响应。非平稳激励用**演变功率谱** $\mathbf G_{FF}(\omega,t)$ 描述（区别于平稳谱 $\mathbf S_{FF}(\omega)$）。一种常用的标量调制模型为 $\mathbf G_{FF}(\omega,t)=|A(\omega,t)|^2\,\mathbf S_{FF}^0(\omega)$，其中 $\mathbf S_{FF}^0$ 为基准平稳谱、$A(\omega,t)$ 为时变调制（包络）函数；该模型用于描述调制型非平稳过程，不代表所有一般非平稳过程。采用**时域虚拟激励法**<sup>[3]</sup>（演变谱与非平稳处理的原理见 2.1.4）：将基准谱逐频点分解为 $\mathbf S_{FF}^0(\omega)=\sum_k\mathbf s_k(\omega)\,\mathbf s_k^{\mathsf H}(\omega)$，构造时变虚拟激励 $\tilde{\mathbf F}_k(\omega,t)=A(\omega,t)\,\mathbf s_k(\omega)\,e^{i\omega t}$，通过标准时域求解服务接口逐步求解 $\tilde{\mathbf u}_k(\omega,t)$（因结构存在记忆效应，不能将平稳公式逐时刻照搬），重构输出演变功率谱 $\mathbf G_{uu}(\omega,t)=\sum_k\tilde{\mathbf u}_k\tilde{\mathbf u}_k^{\mathsf H}$。采用“演变功率谱沿双边角频率积分直接得到时变方差”的谱归一化约定，则

$$\sigma_u^2(t)=\int_{-\infty}^{+\infty}G_{uu}(\omega,t)\,\mathrm d\omega,\qquad \sigma_u(t)=\sqrt{\sigma_u^2(t)},\qquad u_{\mathrm{rms}}(t)=\sqrt{\mu_u^2(t)+\sigma_u^2(t)}$$

零均值时有 $u_{\mathrm{rms}}(t)=\sigma_u(t)$。若采用单边谱或其他傅里叶变换约定，则同步调整积分区间和归一化系数。提供算例测试报告，测试中采用平稳极限解和蒙特卡洛参考结果进行交叉校核。

**▲15 非平稳随机振动分析支持响应概率密度函数的计算。**（提供算例测试报告）完全响应。对于高斯激励且初始条件为确定性或高斯的线性系统，响应在每个固定时刻仍为高斯分布，其边缘概率密度由 $\mu_u(t)$、$\sigma_u(t)$ 直接派生（零均值时 $\mu_u=0$）：

$$p_u(x,t)=\frac{1}{\sqrt{2\pi}\,\sigma_u(t)}\exp\!\left[-\frac{(x-\mu_u(t))^2}{2\sigma_u^2(t)}\right]$$

此为高斯闭式后处理，无需另解动力学方程。对于线性系统下的非高斯激励，在已给定输入概率模型、样本数据或样本生成规则的条件下，可通过激励样本系综生成、批量线性时域求解和核密度估计计算 $p_u(x,t)$；仅有功率谱不足以唯一确定非高斯过程的概率分布。需注意：固定时刻的边缘密度不等于整个时段最大值的分布，峰值/越界概率另需极值理论。提供算例测试报告（高斯情形以 ▲14 求得的 $\sigma_u(t)$ 代入正态闭式、与解析分布比对；非高斯情形以大样本蒙特卡洛的经验分布/直方图校核核密度估计结果）。

#### （四）响应谱分析算法（▲16–▲20，完全响应）

先由模态 $(\omega_i,\boldsymbol\phi_i)$、阻尼比 $\xi_i$、质量阵 $\mathbf M$ 与地震输入方向向量 $\mathbf r$ 计算模态参与系数

$$\Gamma_i=\frac{\boldsymbol\phi_i^{\mathsf T}\mathbf M\mathbf r}{\boldsymbol\phi_i^{\mathsf T}\mathbf M\boldsymbol\phi_i}$$

此为通用归一化下的表达式；若振型按单位模态质量归一化，则分母 $\boldsymbol\phi_i^{\mathsf T}\mathbf M\boldsymbol\phi_i=1$，退化为 $\Gamma_i=\boldsymbol\phi_i^{\mathsf T}\mathbf M\mathbf r$，与 ▲10 频响所采用的归一化约定一致。

以位移反应谱为例，第 $i$ 阶模态对指定自由度 $p$ 的位移峰值贡献为

$$r_{pi}^{(u)}=\Gamma_i\,\phi_{pi}\,S_d(\omega_i,\xi_i)$$

其中，$S_d(\omega_i,\xi_i)$ 为位移反应谱在第 $i$ 阶固有频率和阻尼比处的纵坐标，$\phi_{pi}$ 为第 $i$ 阶振型在自由度 $p$ 处的分量。目标为加速度、内力或应力时，采用相应反应谱量并通过对应结果恢复算子计算，不能直接套用位移峰值公式。对同一指定响应量，将各阶峰值贡献简记为 $r_i$，再按以下方法组合得到总峰值估计 $R$。由于各阶峰值通常不在同一时刻发生，组合结果一般不保留正负号和峰值发生时刻。

**▲16 响应谱分析支持单点、多点激励施加。** 完全响应。**单点（一致）激励**下全结构受同一地面运动，按组前推导的模态参与系数 $\Gamma_i$ 与反应谱直接计算各阶峰值贡献。**多点（多支座）激励**下各支座运动不同，结构总响应分解为两部分：**拟静力分量**——结构静态跟随各支座位移 $\mathbf u_g$ 产生的响应 $\mathbf u_{\mathrm{st}}=\mathbf R\,\mathbf u_g$，$\mathbf R$ 为拟静力影响矩阵（其各列为结构对相应支座单位位移的静态响应，单支座一致激励时退化为方向向量 $\mathbf r$；此处 $\mathbf R$ 为矩阵，与下文模态组合的总峰值标量 $R$ 不同）；**动力分量**——由各支座运动激发的惯性振动响应，按各支座反应谱及对应影响向量列分别做模态叠加。两类分量及不同支座贡献的组合需计入支座运动的空间相关性，具体组合规则依据选定规范（如各支座独立或给定相干模型）配置，不直接套用单点、单方向激励的模态参与系数公式。

**▲17 支持 SRSS 叠加法。**（提供算法原理描述与算例测试报告）完全响应。对于同一指定响应量，令 $r_i$ 为第 $i$ 阶模态的峰值贡献，则组合后的总峰值估计为 $R=\sqrt{\sum_i r_i^2}$。该方法适用于固有频率分离较好、各阶峰值响应相关性较弱的情形（统计假设见 2.1.5<sup>[4]</sup>）。

**▲18 支持 NRL 叠加法。**（提供算法原理描述与算例测试报告）完全响应。NRL（美国海军研究实验室法，Naval Research Laboratory）<sup>[11]</sup> 取绝对值最大的模态贡献与其余模态的 SRSS 结果相加。令 $k=\arg\max_i|r_i|$，则

$$R_{\mathrm{NRL}}=|r_k|+\sqrt{\sum_{i\ne k}r_i^2}$$

该方法属于工程中使用的保守组合方法。

**▲19 支持 GRP 叠加法。**（提供算法原理描述与算例测试报告）完全响应。GRP（分组法，Grouping Method）<sup>[6]</sup> 将频率相近的模态（如规范常用的相邻频率相差 10% 以内）划入同一组 $\{G_q\}$，组内按模态贡献绝对值相加、组间按 SRSS 组合：

$$R=\Big[\sum_i r_i^2+2\sum_q\sum_{\substack{i<j\\ i,j\in G_q}}|r_i|\,|r_j|\Big]^{1/2}$$

第一项为全部模态的 SRSS，第二项仅对同组模态补入交叉贡献（同组仅两阶时退化为绝对值相加 $|r_i|+|r_j|$），从而在 SRSS 基础上计入密集模态间的相关。工程实现中明确并记录所采用的分组准则和阈值。

**▲20 支持 CQC 叠加法。**（提供算法原理描述与算例测试报告）完全响应。基于模态相关系数 $\rho_{ij}$ 的完全二次组合 $R=\sqrt{\sum_i\sum_j \rho_{ij}\, r_i r_j}$，是 SRSS 在密集模态相关情形下的推广<sup>[4,5]</sup>。在线性结构、经典阻尼及白噪声输入假设下，令频率比 $\beta_{ij}=\omega_j/\omega_i$，一种常见的相关系数形式为：

$$\rho_{ij}=\frac{8\sqrt{\xi_i\xi_j}\,(\xi_i+\beta_{ij}\xi_j)\,\beta_{ij}^{3/2}}{(1-\beta_{ij}^2)^2+4\xi_i\xi_j\beta_{ij}(1+\beta_{ij}^2)+4(\xi_i^2+\xi_j^2)\beta_{ij}^2}$$

该方法用于考虑密集模态之间的相关性。工程实现中明确所采用的规范版本、频率比定义、阻尼模型和适用假设；采用其他规范或输入假设时，按相应规定选取相关系数。

## 五、技术先进性说明

### 5.1 算法体系的先进性

**（1）虚拟激励法（PEM）—— 随机振动的高效算法。** PEM 通过谱分解将随机响应谱计算转化为确定性简谐计算序列，能够在输入谱分解中保留多点激励的相关信息。本方案拟以 PEM 作为功率谱分析的核心算法，并采用时域虚拟激励与逐步积分相结合的技术路线处理非平稳随机振动问题。相关算法按照“数学模型明确、接口边界清晰、标准算例验证、工程算例复核”的方式实施。

**（2）功率谱算法体系覆盖不同适用场景。** CQC 用于考虑模态交叉相关影响，SRSS 适用于模态响应相关性较弱的情形，PEM 适用于通过确定性频响计算处理单点或多点相关激励；三种方法可在适用条件重叠的算例中开展交叉校核。

**（3）响应谱叠加方法类型完整。** SRSS、NRL、GRP、CQC 四种组合方法分别提供忽略模态相关、保守组合、密频分组和相关系数组合等不同处理路径；具体适用范围、分组准则和相关系数按照项目选定规范及分析假设配置。

**（4）非平稳随机振动的工程化技术路线。** 采用演变功率谱、时域虚拟激励与逐步积分相结合的方案，面向时变均方根和响应概率密度等招标指标开展实现与算例验证。

**（5）支撑接口具有开放性。** 模态、频率响应和时域积分能力均通过标准服务接口调用，可接入采购人现有求解器、A/B 包模块或测试支撑组件，使随机振动算法与具体底层求解实现解耦，便于后续替换、扩展和性能优化。

#### 5.4.1 工程结构与构建系统

模块以**标准 C++（如 C++17）**实现核心算法，对外提供 C/C++ 头文件接口与 Python 绑定，便于嵌入采购人求解器框架并支持科研人员快速验证与二次开发。源代码按职责分层组织为统一目录结构，接口、实现、测试、算例、脚本与文档边界清晰：

```text
structural-random-vibration/
├── CMakeLists.txt        # 顶层构建入口
├── cmake/                # 构建辅助模块与工具链文件（含国产平台）
├── include/              # 对外公开头文件（库接口）
├── src/                  # 三类算法库与支撑接口实现
├── tests/                # 单元测试与回归测试
├── examples/             # 标准算例（解析解 / 基准 / 工程规模）
├── benchmarks/           # 基准数据与结果比对
├── python/               # Python 绑定
├── scripts/              # 构建、测试、打包等自动化脚本
├── docs/                 # 算法原理手册、用户手册、接口文档
└── third_party/          # 锁定版本的开源依赖
```

采用 CMake 作为统一构建系统，out-of-source 构建，通过工具链文件（toolchain file）支持本地编译与面向国产平台的交叉编译；第三方依赖仅限可再分发的开源数值库，随源码一并交付并锁定版本，不引入任何受限商业组件，保证采购人在隔离环境下可完整复现构建。

#### 5.4.2 安装与编译流程

提供跨平台一键构建脚本（Linux 下 `build.sh`、Windows 下 `build.ps1`），封装"环境检查 → 配置 → 编译 → 测试 → 安装"全流程：

（1）环境检查脚本探测 C++ 编译器（兼容 GCC、Clang 等主流编译器及国产平台编译器）、CMake 版本、依赖库与目标平台（含国产 CPU/操作系统），缺项即时报错并给出处置建议；

（2）配置与编译采用标准 `cmake -S . -B build` 与 `cmake --build build`，支持 Debug/Release 及交叉编译选项；

（3）`cmake --install` 产出规范化交付物——静态/动态库、对外头文件、Python wheel 包与文档，安装目录结构标准化，便于嵌入采购人求解器框架。

最小依赖清单与各平台编译说明随源码以文档形式提供，确保采购人无需我方在场即可独立完成从源码到可用库的全过程。

#### 5.4.3 自动化管理脚本

将日常研发与维护操作全部脚本化、一键化，降低采购人接手后的维护门槛：

脚本统一入参与日志规范，既可独立运行，也可由持续集成流水线统一调度。

#### 5.4.4 持续集成与回归保障

全部算例纳入自动化回归测试集（与 2.5 节质量保证体系一致）。每次代码提交触发持续集成流水线：在多平台编译矩阵（含国产软硬件平台）上完成构建，并全量运行单元测试与三级算例回归；任一环节失败即阻断合入并留存日志。针对不同编译器、稀疏数值库及国产平台可能引入的浮点微差，回归比对按设定数值容差判定通过（而非要求逐位二进制相等），并对跨平台关键结果差异设阈监控，避免数值微差在验收阶段放大为争议。该流水线同时构成可重复执行的验收测试基础——交付时可在采购人环境下复跑同一套脚本与算例，使验收"有据可依、可复现"。

### 5.5 能力演进路线（接口预留的后续增强）

本方案首交版本完整覆盖招标全部 3 个 ★ 与 20 个 ▲ 指标，先进性以工程化交付体系为核心（见 5.1–5.4）。与此同时，分层解耦架构与标准支撑接口（见 2.3、2.4 节）为后续能力演进预留了平滑接入点——下述方向可在不重构上层算法、不改动已交付接口契约的前提下逐步纳入，使 C 包从"满足招标的随机振动模块"成长为"可持续演进的随机振动分析基础设施"。**本节为架构可扩展性的体现与演进规划，非首交承诺范围。**

**第一阶段（首交，合同范围）。** 线性结构、高斯激励假设下的平稳随机振动（CQC/SRSS/PEM 功率谱与均方根）、非平稳随机振动（演变谱 + 时域虚拟激励的时变均方根与概率密度）、响应谱四种模态组合（SRSS/NRL/GRP/CQC），并配套三级验证、自动回归与中文文档——即本说明第三章逐条响应的全部内容。

**第二阶段（非高斯 / 高阶统计）。** 在既有演变谱与样本统计接口上扩展非高斯随机过程建模，如演变双谱等高阶谱表示与样本系综统计<sup>[12]</sup>，覆盖偏离高斯假设的强非平稳激励；▲15 已预留的样本统计通道为此提供入口。

**第三阶段（概率代理 / 大规模不确定性量化）。** 面向频繁参数扫描、敏感性分析、脆弱性与可靠性评估等重复求解场景，叠加多项式混沌展开（PCE）、稀疏概率配置等代理模型层<sup>[13,14]</sup>，将重复随机仿真的成本前移到代理建模阶段，显著降低大规模不确定性量化的计算开销。

**横向增强。** 非经典阻尼（复模态）下的随机响应、概率风险分析等方向，亦可通过同一支撑服务接口与结果恢复层接入，不影响既有平稳/非平稳/响应谱算法。

各阶段均以"接口稳定、向后兼容"为约束推进：后续增强以新增算法库或服务的方式实现，复用既有数据协议与求解服务接口，不破坏已交付版本的兼容性。
